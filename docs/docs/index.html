This Api represents programmatic access to the data of the game Hades' Star. Getting given modules, ships and other data.

GET /modules

Returns all modules

GET /modules/{module.name}

Return a specific module by name

GET /ships

Return all ships, including Cerberus ships with their unique module

GET /ships/{ship.name}

Return a specific ship by name
