GET /modules/{module.name}

Return a specific module by name

GET /modules/types/{type.name}

Return an array of modules in a category according to the order in the game

GET /ships/{module.name}

Return a specific ship by name

GET /ships/types/{type.name}

Return an array of ships in a category according to the order in the game
